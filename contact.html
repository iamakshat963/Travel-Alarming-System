This system is Created for CAP463-Capstone Project in the Guidance of Yamini Bhardwaj
This system uses Google Map APi (Maps javascript APi,Place APi,Directions APi,Distance Matrix API). The Maps JavaScript API is used to embed interactive maps directly into web pages. The Places API is used to search for places, retrieve details. The Direction APi is used to get a appropriate route from A to B(if exists). The Distance Matrix API is used find the total Distance and Time to reach the Destination.
The system works on the principle of refreshing the data collected by Direction and Distance Matrix APi and show the following of the webpage. if the distance to reach reaches 5km a new webpage will pop as a alarm(mp3) and it will start.
